# Speech Recognition Tuning Tool

## Overview

While working at Intervoice, Inc. (2001-2007), I designed and developed the Universal Tuning Tool.

Intervoice's Professional Services department relied on this tool to measurably improve speech application performance for many customers, including:

- Geico
- T-Mobile
- Chase
- US Bank
- Continental Airlines
- the IRS
- Baylor Medical
- Sirius Radio

Features:

- multiple platforms: IQTalk, VXML, HP OCMP
- multiple ASR vendors: Nuance, Speechworks, Microsoft
- multiple languages: English, Spanish, Portuguese
- call review and transcription
- offline grammar experiments.
