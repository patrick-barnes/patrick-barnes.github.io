# Cultural Knowledge Graph

## Overview
A prototype system for representing cultural beliefs, practices, narratives, and traditions as structured knowledge linked to textual sources.

## Why This Matters
Cultural and religious knowledge is difficult to model well because meaning depends on relationships, context, interpretation, and source grounding.

## What I'm Exploring
- entities and relationships
- source-linked assertions
- graph structure vs vector retrieval
- AI-assisted querying and synthesis

## Possible Technologies
- Python
- Neo4j
- embeddings / vector search
- Markdown or JSON source corpora

## Status
Concept and architecture exploration.

## What This Demonstrates
This project reflects my interest in knowledge systems, AI, culture, and structured learning.
