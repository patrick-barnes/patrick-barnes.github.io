## Overview

An app that explores Final Fantasy VII game data

## Status

Publicly and freely available

- Website: <a href="https://picklejar76.github.io/kujata-webapp/">https://picklejar76.github.io/kujata-webapp/</a>
- Source:
  - <a href="https://github.com/picklejar76/kujata">kujata</a>: translates game data to JSON and glTF
  - <a href="https://github.com/picklejar76/kujata-data">kujata-data</a>: the game data in JSON and glTF
  - <a href="https://github.com/picklejar76/kujata-webapp">kujata-webapp</a>: the app to view the game data
